{% markdown %} ## Text container Text containers center and restrict the size of content to a nice, readable width. These containers are also responsive down to mobile. It is recommended that the page width vary for a nicer look. You'll want to add some classes to your images or they will not scale responsively. ![Alt text for a large, random image](https://picsum.photos/640/240){.ui .fluid .rounded .image} {% endmarkdown %}
{% markdown %} ## Container spacing Vertical spacing is also really important to maintain. The best way to do this is with padded containers. This provides a break in the content to give the reader some reference point in the document. {% endmarkdown %}
